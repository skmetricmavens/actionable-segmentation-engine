# Getting Started

Welcome to the Actionable Segmentation Engine! This section will help you get up and running quickly.

## Prerequisites

Before you begin, ensure you have:

- **Python 3.11+** installed
- **pip** or **uv** for package management
- (Optional) **Claude API key** for real LLM integration

## Installation Options

=== "pip"

    ```bash
    pip install -r requirements.txt
    ```

=== "uv"

    ```bash
    uv pip install -r requirements.txt
    ```
